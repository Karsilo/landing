## Adding PDF Files

1. Create your question PDFs using LaTeX, Word, or your preferred tool
2. Save questions as `{module-name}-questions.pdf`
3. Save answers as `{module-name}-answers.pdf`
4. Place them in the appropriate course directory

## File Naming Convention

- Use lowercase
- Use hyphens for spaces
- Follow the pattern: `{module-name}-{type}.pdf`
- Types: `questions` or `answers`

## Creating Placeholder PDFs

You can create simple placeholder PDFs using:
- Online tools like Canva or Google Docs
- LaTeX
- Microsoft Word/Pages
- Any PDF editor

Each PDF should contain:
- Course name and module name
- Question numbers
- Adequate space for working
- Answer key (for answer PDFs)
